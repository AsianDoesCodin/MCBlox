McBlox Project Status — April 7, 2026

- Player heartbeat: 60s ping while playing, live player counts

- Logo: Everywhere (website nav/hero, launcher sidebar, Tauri icons, favicon)
- GitHub Release v0.1.0: .exe + .msi installers uploaded
- END-TO-END MC LAUNCH: Tested with Payter (1.20.1 Forge 47.4.10). Full pipeline:
  MediaFire download → zip extract/flatten → MC client + libs + assets → Forge installer
  processors (client-srg, client-extra, patched) → Java 17 auto-download → natives extraction
  → correct JVM/game args → Minecraft launches with mods!
