McBlox Project Status — April 7, 2026

- Space theme: Neon green (#00e676) accents, dark galaxy background, cyan highlights

  client JAR + libraries + assets download, classpath builder, launch args, auto-join server

- Custom McBlox icon: Installer, taskbar, exe, favicon
- SVG search button (replaced emoji)
- Storage management: Settings shows breakdown, "Clear Game Cache" button

BUILD & RELEASE:
- Dev:   cd launcher && npm run tauri dev
- Build: $env:TAURI_SIGNING_PRIVATE_KEY = (Get-Content ~/.tauri/mcblox2.key -Raw)
         $env:TAURI_SIGNING_PRIVATE_KEY_PASSWORD = ""
         cd launcher && npx tauri build (press Enter at password prompt)
- Release: Create latest.json from .nsis.zip.sig, gh release create with .exe, .nsis.zip, latest.json
